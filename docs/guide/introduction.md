# 项目介绍

为帮助社区开发者快速搭建风格统一、易于扩展的移动端应用，**TDesign Mobile Vue Apply** 模板应运而生。

## 技术栈

项目以 **TDesign Mobile Vue** 为基础，采用现代化的前端技术栈：

- **Vue 3** - 渐进式前端框架
- **Vite** - 快速的构建工具
- **TypeScript** - 类型安全的 JavaScript

## 功能特性

项目拥有以下核心页面：

### 主要页面

- **首页** - 展示热门推荐和活动列表
- **我的** - 个人信息与报名活动查看
- **活动详情** - 时间/地点/票价/嘉宾/评价等
- **报名与结果** - 订单确认、报名人信息填写、购买结果页
